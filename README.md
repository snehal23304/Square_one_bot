# 🤖 Square One Bot

A beginner-friendly bot project designed to start from "square one" and build core functionality step by step. This project serves as a foundation for learning automation, bot development, and structured coding practices.

---

## 🚀 Features

- 🧠 Simple and modular design
- ⚡ Easy to extend and customize
- 🔌 Beginner-friendly structure
- 📦 Clean and readable codebase

---

## 🛠️ Tech Stack

- Language: Python ,C/C++
- Tools: Git, VS Code ,Arduino IDE , RasperryPi , Arduino Mega
